`
  setup                初始化（第一次部署才會用到）
  update               更新到最新的 commit
  revert [n]           回复到上一次的 deployment
  curr[ent]            输出目前上线中的 commit 
  prev[ious]           输出上一次部署的 commit
  exec|run <cmd>       执行指定的指令
  list                 列出包含目前，以及之前所部署的 commit
  [ref]                部署到指定的 ref
